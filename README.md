# Homebrew Tap for ws-tools

This tap contains Homebrew formulas for [ws-tools](https://github.com/0xthc/ws-tools).

## Installation

```bash
brew tap 0xthc/tap
brew install ws
brew install texplore
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `ws` | Workspace CLI for git worktrees with tmux layouts |
| `texplore` | Terminal file explorer with git integration |

## Updating

```bash
brew update
brew upgrade ws texplore
```
